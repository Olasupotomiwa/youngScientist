  2. Using the a proportion and sample size

  4. Using two proportions and sample size from two different population

Note: the population mean or proportion is always given

How to Calculate the Zstatistics of a population

Here we will learn how to calculate Zstatistics when dealing with hust a population. This is of two category

Sample mean, standard deviation and sample size are given.

$$Z_{cal}=\frac{\bar X-μ}{σ/\sqrt{n}}$$ $$\bar X = sampleMean$$ $$σ = Standard Deviation$$ $$ n = sample size$$

It should be noted that the denominator can be replace by Standard Error. since $$S.E=\frac{σ}{\sqrt{n}}$$

when proportion and sample size are given

$$Z_{cal}=\frac{P-P_0}{\sqrt{\frac{P_0(1-P_0)}{n}}}$$ $$ P_0 = population Proportion$$ $$P = SampleProportion$$

It should be noted that the denominator can be replace by Standard Error. since $$S.E=\sqrt{\frac{P_0(1-P_0)}{n}}$$
